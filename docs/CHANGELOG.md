# Changelog

## 1.4.1 — 2026-07-14

### Редизайн GUI
- Новая палитра: тёмный фон `#0D1117`, карточки `#161B22`, мягкие violet `#7C74FF` + cyan `#38BDF8` акценты — не режет глаза; второстепенный текст ярче (контраст ≥4.5:1), мелкий шрифт 10→11
- Главная с нуля: витрина GGSELL во всю ширину со статистикой в ряд, ниже — сервисы по очереди горизонтальными карточками
- Карточки кликабельны целиком (hover — плавная подсветка рамки акцентом), большие кнопки «Открыть» убраны
- Тонкие декоративные полоски без скруглений — убраны «пиксельные» зазубрины отрисовки CustomTkinter
- Логи GUI теперь пишутся в `automation.log` (единый журнал: приложение + прогоны); экран «Логи» читает из файла
- Тема осветлена (slate вместо почти-чёрного: фон `#161C28`, карточки `#1E2634`) — остаётся тёмной, но не «яма»
- Колесо мыши скроллит список под курсором (Tk на Windows шлёт колесо виджету с фокусом — исправлено глобальным обработчиком)
- Скроллбар скрывается автоматически, когда контент помещается; главная — статичная, без скролла и анимаций входа
- Исправлено «уезжание» текста и блоков при скролле (колесо крутило canvas отрисовки CTk-виджетов) и «залипание» отступа/невидимых карточек при быстрой смене вкладок
- Статус в sidebar одной строкой: «Простой · Профили: N · приложение»; панель «Активность» без лишних кнопок, поле до низа; вкладка «Логи» — на всю высоту окна
- Sidebar заново: секции «Разделы»/«Система» с подписями, у пунктов плитки-иконки с буквой и цветной подсветкой активного; «Логи» и «Настройки» — пункты навигации, а не кнопки внизу; подсветка активного пункта обновляется при любом переключении (раньше индикатор «залипал»)
- Компактная плотность: отступы 24→20/16, кнопки 38→34, радиусы 18→14, sidebar 248→232, заголовки 28→24
- Плавные переходы страниц: слайд с ease-out + каскадное «проявление» карточек — без пересборки виджетов, не лагает
- Все анимации цветов — с ease-out cubic вместо линейных
- Кэш шрифтов: один `CTkFont` на (семейство, размер, вес) вместо 130+ копий — быстрее построение страниц, меньше память
- Переключение вкладок: `grid_forget` только предыдущей страницы вместо всех

### Установка и удаление
- «Uninstall SubHub.exe» теперь действительно удаляет приложение: копия установщика с именем «Uninstall…» сама включает режим удаления (раньше без аргумента `/uninstall` открывалось окно установки)
- Новый экран удаления: подтверждение, галочка «также удалить данные и профили» (по умолчанию выкл., с отдельным подтверждением), закрытие запущенного SubHub/pythonw перед удалением
- Деинсталлятор удаляет сам себя и пустую папку установки после завершения
- Папка установки определяется по местоположению деинсталлятора (не только из реестра) — работает при установке в любую папку, например `D:\SubHub`
- Редизайн установщика: тёмная тема, прогресс-бар, безрамочное окно
- В payload добавлен `scripts\_gui_boot.py` (crash-лог при падении GUI)
- Версия Inno Setup берётся из файла `VERSION`



### Надёжность
- VeepN для Flipkart: USA → FR → DE (если США недоступен — не падаем сразу)
- После неудачного базового VPN — перебор стран через `_vpn_reconnect_for_flipkart`
- Sticky cancel / VPN profile registry / Stop Run: `taskkill` дерева процессов

### Windows-приложение
- Ярлыки: рабочий стол + меню «Пуск» (закрепление через «Все приложения»)
- Portable install: `scripts/install_subhub.ps1`, Inno Setup: `scripts/SubHub.iss` + `build_installer.bat`
- Кнопки в Настройках: «В меню Пуск», «Бэкап»

### Продукт / UI
- GGSELL: монитор Вкл/Выкл, превью шаблонов доставки; панель управления только в «Обзор»
- Kling скрыт с главной (`ready: False`)
- Тосты для операций; горячие клавиши: F5 / Ctrl+R (обновить), Ctrl+Shift+S (стоп), Esc (закрыть тосты)
- Бэкап: `scripts/backup_data.py` → `data/backups/subhub_backup_*.zip`
- Health-check селекторов Flipkart: `scripts/test_selector_health.py` (в smoke)
